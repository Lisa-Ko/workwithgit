## Журнал изменений
Ддя просмотра истории коммитов используется команда *git log*. Для этого в терминале с папкой-репозиторием пишим *git log*. В журнале обязательно будут:
* Хэш(номер) коммита
* Дата и время коммита
* Автор коммита
* Текст сщщбщение к коммиту
